## 1. スカルノハッタ国際空港（CGK）

### 1.1 ターミナル構成

| ターミナル | 主な航空会社 | 特徴 |
|-----------|------------|------|
| **Terminal 1** | 国内線LCC（ライオンエア等） | 1A/1B/1Cの3棟。古い |
| **Terminal 2** | 国内線（ガルーダ国内線）＋一部国際線 | 2D/2E/2Fの3棟 |
| **Terminal 3** | **国際線メイン**（ガルーダ国際線、ANA、JAL、シンガポール航空等） | 最新・最大。到着は**Ground Floor** |

> **日本からの直行便はTerminal 3に到着する。** ANA（NH855/NH871）、JAL（JL725）、ガルーダ・インドネシア航空（GA881/GA885）いずれもT3。

### 1.2 到着の流れ（Terminal 3 国際線）

```
飛行機降機
  ↓
ボーディングブリッジ → 到着コンコース
  ↓
【1】All Indonesia Document Verification（デジタル入国カード確認）
  ↓
【2】VOA（ビザ・オン・アライバル）カウンター ★入国審査の手前
  ↓
【3】入国審査（Immigration）— 自動ゲート78台 + 有人カウンター
  ↓
【4】手荷物受取（Baggage Claim）— フライト番号でベルト番号確認
  ↓
【5】税関（Customs）— 緑：申告なし / 赤：申告あり
  ↓
到着ロビー（Arrival Hall）→ SIMカード・両替・ATM・交通手段
```

### 1.3 VOA（Visa on Arrival / 到着ビザ）

| 項目 | 詳細 |
|------|------|
| **対象者** | 日本国籍パスポート保持者（30日以内の観光） |
| **費用** | **IDR 500,000**（約680円 / 約$35 USD） |
| **支払方法** | クレジットカード（Visa/Master）、現金（USD/IDR） |
| **カウンター位置** | 入国審査の**手前右側**。大きな「VISA ON ARRIVAL」の看板あり |
| **必要書類** | パスポート（残存6ヶ月以上）、帰国便または第三国行きチケット |
| **所要時間** | 15〜40分（混雑状況による） |
| **延長** | 30日間の延長可能（現地の入管オフィスで申請、IDR 500,000追加） |
| **オンライン申請** | [Molina](https://molina.imigrasi.go.id/) で事前にe-VOA申請可能。空港でのカウンター待ちを省略できる |

> **重要（2025年8月18日〜）**: **All Indonesia Arrival Card** の事前登録が義務化。到着3日前から [All Indonesia](https://all-indonesia.com/) サイトで登録可能。パスポート情報・滞在先・フライト情報を入力。QRコードが発行される。

### 1.4 SIMカード購入

到着ロビーに複数のSIMカードショップがある。

| キャリア | プラン | 価格 | データ量 | 有効期間 | 備考 |
|---------|-------|------|---------|---------|------|
| **Telkomsel Tourist SIM** | Tourist Prepaid Card | **IDR 150,000** | 25GB + 25分通話 | 30日間 | 最も広いカバー率。ジャワ・バリ全域で安定 |
| **XL Axiata** | Tourist SIM | IDR 100,000〜200,000 | 15GB〜25GB | 30日間 | ジャワ・バリは問題なし。山間部やや弱い |
| **Indosat (IM3)** | Tourist Plan | IDR 100,000〜 | 15GB〜 | 30日間 | コスパ良いが一部地方でカバー弱い |

> **おすすめ: Telkomsel Tourist SIM（IDR 150,000）** — 10日間のジャワ横断ではTelkomselの回線品質が最も安定。空港T3到着ロビーのTelkomselカウンターで購入・アクティベーションまで対応（10〜15分）。パスポート提示必須。

> **eSIM派**: Holafly、Airalo、Nomad等でインドネシアeSIMを事前購入可能。物理SIMの購入が面倒な場合は便利だが、GrabやGoJekの登録にはSMS受信可能なインドネシア番号が必要なので、物理SIMの方が汎用性が高い。

> **IMEI登録**: 海外で購入したスマートフォンを現地SIMで使う場合、税関でIMEI登録が必要。到着時にCustoms（税関）エリアの端末で登録するか、オンラインで事前登録。登録しないと90日後にSIMが使えなくなる。

### 1.5 両替・ATM

#### ATMの場所
- **入国審査前**: 数台のATMあり（ただしVOA支払い用の現金引き出しに使える）
- **手荷物ターンテーブル前**: 数台設置
- **到着ロビー**: BCA、BNI、Mandiri、BRI等の主要銀行ATM多数

#### 両替所
| 場所 | レート | 営業時間 | 備考 |
|------|-------|---------|------|
| 空港到着ロビー両替所 | やや悪い（5〜10%マージン） | 24時間営業の店舗あり | 最低限の両替に留めるべき |
| 空港ATM | 中程度（銀行レート+手数料） | 24時間 | 海外キャッシング対応カード推奨 |
| ジャカルタ市内両替商 | 良い | 店舗による | Dolarindo、Central Kuta等が有名 |

> **推奨両替戦略**: 空港では**ATMで最低限（IDR 500,000〜1,000,000程度）**を引き出す。大額の両替は市内で。WiseデビットカードやSony Bank WALLETなどの海外利用に強いカードがあれば、ATMキャッシングが最もレートが良い。

> **注意**: 空港の両替所では「レートが良い」と言いつつ手数料を別途取る店や、旧紙幣（折れ・汚れ）を混ぜて渡してくる店がある。必ず受け取った紙幣をその場で数え、金額を確認すること。

## 2. 空港からガンビル駅へのアクセス

### 2.1 交通手段比較表

| 手段 | 料金 | 所要時間 | 乗り場 | メリット | デメリット |
|------|------|---------|--------|---------|-----------|
| **空港鉄道（KAI Bandara）** → タクシー | IDR 70,000 + タクシー約IDR 30,000 | 約50分+10分 | T3地下SHIA駅 | 渋滞なし・快適 | 乗り換え必要 |
| **DAMRIバス** | IDR 40,000〜80,000 | 50分〜1.5時間 | 各ターミナル前バス停 | **ガンビル直通**・安い | 渋滞で大幅遅延リスク |
| **タクシー（BlueBird）** | IDR 200,000〜350,000 | 40分〜1.5時間 | 到着ロビー前タクシー乗り場 | 荷物が多い時に楽 | 渋滞時は料金も時間も増大 |
| **Grab / GoJek** | IDR 200,000〜300,000 | 40分〜1.5時間 | Ride-hailing専用ピックアップポイント | アプリで事前に料金確定 | インドネシアSIM推奨 |

### 2.2 空港鉄道（KAI Bandara / スカルノハッタ空港コミューターライン）

2017年開業。空港と市内を**渋滞なし**で結ぶ最も時間が読める手段。

| 項目 | 詳細 |
|------|------|
| **路線** | SHIA駅（空港） → Batu Ceper → Duri → **BNI City（Sudirman Baru）** → Manggarai |
| **料金** | **IDR 30,000**（KAI Bandara Premium）/ IDR 70,000（Executive） |
| **所要時間** | 約46分（BNI City駅まで） |
| **運行時間** | 空港発: 06:20〜23:20 / BNI City発: 04:51〜21:51 |
| **頻度** | 約30分間隔 |
| **支払方法** | クレジットカード、デビットカード、電子マネー（現金不可） |
| **購入場所** | SHIA駅の自動券売機 or Railinkアプリ |
| **設備** | エアコン完備、無料WiFi、USB充電ポート、快適な座席 |

#### SHIA駅（空港駅）への行き方
1. T3到着ロビーから「Airport Train」の案内表示に従う
2. スカイトレインに乗車（T1/T2からの場合）→ SHIA駅へ接続
3. T3からは直接徒歩でSHIA駅へアクセス可能

#### BNI City駅からガンビル駅へ

BNI City駅とガンビル駅は**約3km離れている**（徒歩30〜40分）。以下の方法で移動：

| 手段 | 料金 | 所要時間 | 備考 |
|------|------|---------|------|
| **タクシー / Grab** | IDR 20,000〜40,000 | 7〜15分 | 最も早い。BNI City駅前にタクシーあり |
| **TransJakarta BRT** | IDR 3,500 | 15〜20分 | Tosari→Balai Kota下車。安いが路線を調べる必要あり |
| **KRL コミューターライン** | IDR 3,000〜5,000 | 約34分 | Manggarai経由で遠回り（非推奨） |
| **バイクタクシー（Grab Bike）** | IDR 10,000〜15,000 | 10分 | 荷物少なければ最速最安 |

> **実用的な推奨ルート**: 空港鉄道でBNI City → Grabタクシーでガンビル駅（合計約IDR 100,000〜110,000、所要約1時間）

### 2.3 DAMRIバス（ガンビル直通）

空港とガンビル駅を**直通**で結ぶ国営バスサービス。乗り換えなしでガンビル駅前に到着するのが最大のメリット。

| 項目 | 詳細 |
|------|------|
| **路線名** | JAC Gambir（DAMRI Airport → Gambir） |
| **料金** | **IDR 40,000〜80,000**（時期・情報源により異なる。最新はIDR 80,000前後） |
| **所要時間** | 50分〜1.5時間（交通状況による。渋滞時は2時間以上の可能性） |
| **運行時間** | 空港発: 06:00〜24:00 / ガンビル発: 03:00〜20:00 |
| **頻度** | 約15〜60分間隔（時間帯による） |
| **乗り場** | 各ターミナルの「Bus」案内表示に従う。ターミナル前のバスベイ |
| **チケット** | カウンターで購入 or 車内で運転手に支払い。予約不要 |

> **注意**: ジャカルタの渋滞は世界最悪レベル。特に**朝7:00〜10:00**、**夕方16:00〜20:00**は市内移動に通常の2〜3倍の時間がかかる。列車の出発時間が決まっている場合、DAMRIバスでは時間が読めないリスクがある。

### 2.4 タクシー

| 項目 | 詳細 |
|------|------|
| **Blue Bird（ブルーバード）** | 最も信頼できるタクシー会社。メーター制。水色の車体 |
| **料金目安** | IDR 200,000〜350,000（メーター＋高速代IDR 25,000前後） |
| **所要時間** | 40分〜1.5時間 |
| **乗り場** | T3到着ロビー前のタクシー乗り場。行き先を告げてから配車 |
| **支払い** | メーター料金＋高速道路通行料。現金 or カード |

> **警告**: 空港で「タクシー？」と声をかけてくる客引きは**非公式タクシー**の可能性が高い。必ず公式タクシー乗り場から乗車すること。Blue Bird（青い鳥マーク）以外は基本的に避ける。偽Blue Birdも存在するため、公式アプリ「My Blue Bird」で配車するのが最も安全。

### 2.5 Grab / GoJek

| 項目 | 詳細 |
|------|------|
| **料金目安** | IDR 200,000〜300,000（距離＋高速代。アプリで事前表示） |
| **ピックアップ場所** | 各ターミナル指定のRide-hailingピックアップポイント（案内表示あり） |
| **決済** | Grab: 国際クレカ対応 / GoJek: 国際クレカ非対応（現金 or GoPay） |
| **所要時間** | 40分〜1.5時間 |

> **Grab vs GoJekの選び方**:
> - **Grab**: 国際クレジットカードが使える。旅行者にはGrabの方が圧倒的に使いやすい。
> - **GoJek**: 国際クレカ非対応。ただしGoPay（電子ウォレット）に国際Visa/Masterをリンクして使える。現金払いも可能。
> - **両方インストール推奨**: 料金比較ができる。Grabが使えないエリアでGoJekが使えることも。

### 3.4 チケット購入方法

#### 方法1: Traveloka / Tiket.com（外国人推奨）

| プラットフォーム | 特徴 |
|----------------|------|
| **Traveloka** | 国際クレカ対応。日程検索・座席指定可能。英語対応。最も使いやすい |
| **Tiket.com** | 国際クレカ対応。**Whoosh高速鉄道のチケットはTiket.comのみ**で購入可能 |
| **12go.asia** | 英語に完全対応。手数料がやや高いが確実に予約できる |

#### 方法2: KAI Access アプリ（公式）

- PT KAI（インドネシア国鉄）の公式アプリ
- パスポート番号で登録可能だが、**NIK（16桁の国民ID番号）を要求されてエラーになる**報告多数
- **国際クレジットカード非対応** — 事実上、外国人は使いにくい
- 使える場合: e-boardingパスが発行され、出発2時間前からダウンロード可能

#### 方法3: 駅窓口（当日購入）

- ガンビル駅1階のチケットカウンターでパスポート提示して購入
- 人気路線・時間帯は売り切れリスクあり
- **当日購入の場合、出発の3時間前までには駅に到着推奨**

> **推奨**: **Travelokaで事前予約**がベスト。人気時間帯（朝7〜9時台）は数日前に売り切れることも。予約は出発の30〜90日前から可能（列車による）。

### 3.5 乗車の流れ

```
1. 駅到着 → メインエントランスからセキュリティチェック（荷物X線）
  ↓
2. チケット確認（紙チケット or e-boarding pass + パスポート）
  ↓
3. 待合室へ → Executive Lounge（エグゼクティブクラスの場合）
  ↓
4. 出発15〜30分前にホームへの案内アナウンス
  ↓
5. ホーム → 号車番号（Kereta）と座席番号（Nomor Tempat Duduk）を確認して乗車
```

## 4. ジャカルタ→バンドン 列車ガイド

### 4.1 選択肢の比較

| **Whoosh（高速鉄道）** | **約35分** | ハリム | テガルアル | 単一クラス3種 | IDR 250,000〜600,000 |

## 8. 安全情報

### 8.1 一般的な安全レベル

ジャカルタは東南アジアの大都市として標準的な安全レベル。常識的な注意を払えば大きな問題はない。ただし以下の点に注意。

### 8.2 よくある詐欺・トラブル

| 詐欺の種類 | 手口 | 対策 |
|-----------|------|------|
| **非公式タクシー** | 空港で「タクシー？」と声をかけてくる。メーターなし。高額請求 | 公式タクシー乗り場のみ利用。Blue Bird or Grab |
| **偽Blue Bird** | Blue Birdに似た外装のタクシー。メーター細工あり | 公式アプリ「My Blue Bird」で配車。車体の「Blue Bird Group」ロゴ確認 |
| **両替詐欺** | レートが良すぎる両替所。計算を誤魔化す。紙幣を抜く | 空港では銀行系ATMを利用。市内では評判の良い両替商のみ |
| **スリ・置き引き** | 混雑した市場・公共交通機関でポケットやバッグを狙う | 前ポケット使用。バッグは体の前に。スマホは手に握りしめない |
| **ガイド詐欺** | 「無料で案内する」と近づいてくる人。最後に法外な料金請求 | 公式ガイドのみ利用。断る時は「Tidak, terima kasih」（ノーサンキュー） |
| **物乞い組織** | 子供や身体障害者を使った組織的な物乞い | 現金を渡さない方が良い（組織の資金源になる） |
| **ぼったくりレストラン** | メニューに価格なし。食後に法外な請求 | 事前に価格確認。駅構内やチェーン店が安全 |

### 8.3 注意が必要なエリア

| エリア | リスク | 備考 |
|--------|-------|------|

### 8.4 交通安全

| 項目 | 詳細 |
|------|------|
| **横断歩道** | 車は止まらないことが多い。歩行者用信号があっても安全確認必須 |
| **バイク** | 歩道をバイクが走ることがある。常に周囲に注意 |

### 8.5 健康・衛生

| 項目 | 詳細 |
|------|------|
| **水道水** | **絶対に飲まない**。ペットボトル水を購入（IDR 3,000〜5,000/500ml） |
| **氷** | 高級レストラン・カフェは問題なし。屋台の氷はリスクあり |

## 9. 天候・気候・服装

### 9.1 2〜3月の気候

ジャカルタは**熱帯モンスーン気候**。2〜3月は**雨季の真っただ中**。

| 項目 | 2月 | 3月 |
|------|-----|-----|
| **最高気温** | 31℃ | 31℃ |
| **最低気温** | 24℃ | 25℃ |
| **降水量** | 326mm（年間最多月） | 220mm |
| **降雨日数** | 14〜18日 | 14〜16日 |
| **湿度** | 80〜90% | 80〜85% |
| **UV指数** | 非常に高い（11+） | 非常に高い（11+） |

### 9.2 雨の特徴

- **スコール型**: 1日中降り続くのではなく、午後を中心に1〜3時間の激しいスコール
- **朝は晴れ**: 午前中は晴れることが多い。観光は午前に済ませるのが賢い
- **洪水リスク**: ジャカルタは低地のため、大雨時に道路が冠水することがある。冠水した道路は歩かない

### 9.3 推奨服装・持ち物

| カテゴリ | アイテム | 備考 |
|---------|---------|------|
| **上着** | 速乾性の薄手Tシャツ、リネンシャツ | 綿100%は汗で乾きにくい |
| **下** | ショートパンツ or 薄手の長ズボン | モスク訪問時は膝を覆う必要あり |
| **靴** | サンダル + 防水スニーカー | 雨季は足元が濡れやすい。防水性重要 |
| **雨具** | **折りたたみ傘 + レインポンチョ** | 必須。突然のスコールに対応 |
| **日焼け止め** | SPF30以上 | UV指数11+は非常に高い |
| **帽子** | つばの広い帽子 | 直射日光対策 |
| **虫除け** | DEET入りスプレー | デング熱対策 |
| **上着（冷房対策）** | 薄手のカーディガン or パーカー | 列車・バス・モールの冷房が強い |
| **モスク用** | 長ズボン + 肩を覆える上着 | 女性はスカーフも（イスティクラルでは無料貸出あり） |
