## 1. ケタパンとは何か

ケタパン（Ketapang）は、東ジャワ州バニュワンギ県カリプロ地区（Kecamatan Kalipuro）に属するフェリー港町。ジャワ島からバリ島へ渡るフェリーの発着地であるペラブハン・ケタパン（Pelabuhan Ketapang）を中心に、宿泊施設・食堂・コンビニが集まるトランジットタウン。

バニュワンギ市街中心部からは北に約10km（車で20〜30分）。多くの旅行者がイジェン火山やバルラン国立公園の観光後、ここで一泊してから翌朝バリへ渡る。

## 2. フェリー渡航：ケタパン → ギリマヌク 完全ガイド

### 2-1. 運航スケジュール

- **24時間運航**: 毎正時（00:00, 01:00, 02:00 ... 23:00 WIB）に出港
- **日中**: 約15〜20分間隔で頻繁に出港
- **夜間**: 約30分間隔

- 現金払い可能

### 2-4. 乗船の流れ

1. **徒歩旅行者**: ターミナル建物で切符購入 → 待合室 → 乗船案内 → 乗船（比較的スムーズ）
2. **バイク/車**: 専用レーンに並ぶ → 順番に乗船 → 船員の指示に従い駐車 → 上階の客室へ
3. バイクは専用レーンがあり、車より優先的に積み込まれることが多い
4. 車両を固定してから上階の座席エリアへ移動

### 2-5. 船内設備

- **座席**: エアコン付きの座席エリア（レギュラークラス）
- **祈祷室（Mushola）**: あり
- **トイレ**: あり（清潔度はフェリーによりまちまち）
- **カフェテリア**: 軽食・飲料の販売あり
- **充電ステーション**: あり
- **テレビ**: あり
- **船上販売**: ホーカー（行商人）が菓子・飲料・タバコなどを販売

### 2-6. フェリー渡航の実践的Tips

| ポイント | 詳細 |
|----------|------|
| **ピーク時を避ける** | イドゥル・フィトリ（断食明け大祭）、クリスマス・年末年始（Nataru）、学校休暇期間は4〜6時間待ちになることも |
| **ニュピ（Nyepi）に注意** | バリ・ヒンドゥーの新年（2026年は3月19日）。前日からフェリー全面運休。バリ島では24時間外出・照明・移動すべて禁止 |
| **早朝・深夜が空いている** | 日中より深夜〜早朝の方が待ち時間が短い |
| **徒歩が最速** | 車両なしの徒歩旅行者は待ち時間が最短 |
| **小銭を用意** | チケット購入・船内購入用に小額紙幣を準備 |
| **貴重品管理** | フェリー内では荷物から目を離さない |
| **日中渡航が景色良し** | 晴天時はジャワ島の火山群とバリの海岸が美しい |
| **時間に余裕を** | スケジュール通りに運航しないことも多い。前後の予定に1〜2時間のバッファを |

**主要スポット**:

- **パンタイ・バマ（Pantai Bama）**: マングローブ林と白砂のビーチ。シュノーケリング可能な浅瀬。野生のカニクイザルが木の上から海に飛び込む珍しい光景が見られる

**実践的Tips**:

- 乾季（4〜9月）がベストシーズン。サバンナが乾燥して動物が水場に集まる
- 園内にレンタル自転車あり（15〜20kmの舗装路を走行可能）。バイクよりも動物に近づけるとの報告
- 食料・飲料は持参推奨（園内の施設は限定的）
- 日焼け止め・帽子・歩きやすい靴必須
- ガイド雇用: 約Rp 100,000（交渉次第）

## 6. 実用情報

### 6-1. 交通

#### ケタパン駅（Stasiun Ketapang / 旧・バニュワンギ・バル駅）

| 項目 | 詳細 |
|------|------|
| **フェリー港まで** | 約100m（徒歩10分） |
| **主要列車** | プロボワンギ号、タワンアルン号、スリタンジュン号（スラバヤ方面）、パンダンワンギ・ローカル（ジュンベル方面） |
| **スラバヤから** | 約7〜8時間 |
| **ジュンベルから** | パンダンワンギ号で約2.5時間（112km） |
| **マランから** | マラン→ジュンベル→ケタパンと乗り継ぎ |

> **ポイント**: 列車とフェリーの接続が良い。列車でケタパン駅に到着後、徒歩10分でフェリー港へ行ける設計

#### オジェック・Grab

- **Grab / GoJek**: ケタパン地区で利用可能。バイクタクシー（オジェック）と四輪車タクシーの両方
- **港から市街中心部**: バイクで約Rp 15,000〜25,000、車で約Rp 30,000〜50,000
- **港からバルラン国立公園**: 片道Rp 80,000〜150,000程度（交渉次第）
- **Tips**: 港周辺には流しのオジェック（非アプリ）もいるが、Grab/GoJekの方が料金が明朗

#### レンタルバイク/レンタカー

- バニュワンギ市街にレンタルバイク店あり（1日 Rp 60,000〜100,000）
- レンタカー（ドライバー付き）: 1日 Rp 400,000〜600,000
- ケタパン駅周辺でもレンタカーの手配可能

### 6-2. ATM

| **BRI** | Jl. Gatot Subroto, ASDP Ketapang（フェリー港内） |
| **BRI** | SPBU Farly Ketapang, Jl. Gatot Subroto（ガソリンスタンド内） |

### 6-4. 通信

- **Telkomsel**: ケタパン地区は4G LTEエリア。フェリー上でも通信可能（海峡中央部でやや弱くなることあり）
- **SIMカード**: Indomaret/Alfamartで購入可能。パスポート提示必要
- **WiFi**: 主要ホテル（Ketapang Indah等）は無料WiFi。Resto Thalitaも無料WiFi提供

### 6-5. 時差に関する注意

- **ケタパン（ジャワ島）**: WIB（西部インドネシア時間）= UTC+7
- **ギリマヌク（バリ島）**: WITA（中部インドネシア時間）= UTC+8
- **フェリーで海峡を渡ると1時間進む！** ジャワの朝7:00 = バリの朝8:00
- 早朝フェリーで渡る場合、到着後のスケジュールに注意

## 7. 安全情報

### 7-1. 一般的な治安

ケタパン・バニュワンギエリアは**全般的に安全**な地域。凶悪犯罪のリスクは低い。

### 7-2. 注意すべき点

| リスク | 対策 |
|--------|------|
| **スリ・置き引き** | フェリーターミナル、船内、混雑時に注意。バッグは体の前で持つ。必要最低限の現金のみ携帯 |
| **交通事故** | バイク移動時はヘルメット着用必須。夜間のバイク運転は避ける。インドネシアの交通は無秩序に見えることも |
| **客引き・ぼったくり** | 港周辺のオジェック運転手が法外な料金を要求することあり。Grab/GoJekアプリを使えば料金は透明 |
| **偽チケット** | フェリーチケットは公式ブース（ASDP）かFerizyアプリで購入。路上の非公式販売者からは買わない |
| **海峡の天候** | 荒天時はフェリーが遅延・欠航する可能性。雨季（11〜4月）は特に注意 |
| **密造酒** | インドネシア全土で問題。「arak」等の密造酒にメタノール混入の死亡事故が起きている。信頼できる店以外でのアルコール購入は避ける |
| **薬物** | インドネシアでは薬物所持は**死刑**を含む極めて重い刑罰。絶対に関わらない |

### 7-3. 緊急連絡先

| 機関 | 番号 |
|------|------|
| **警察** | 110 |
| **救急** | 118 / 119 |
| **消防** | 113 |
| **ツーリストポリス** | +62 21 526 4073（ジャカルタ本部） |
| **在スラバヤ日本国総領事館** | +62 31 503 0008 |

### 7-4. 健康

- **蚊対策**: デング熱のリスクあり。虫除けスプレー持参（DOT不要）
- **水**: 水道水は飲めない。ミネラルウォーターを購入（Indomaret等で）
- **食事**: 屋台・ワルンの食事は基本的に安全だが、火が通ったものを選ぶのが無難
- **薬局**: Apotek（薬局）がJl. Gatot Subroto沿いに数軒あり
