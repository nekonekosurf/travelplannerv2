## 1. パンガンダランとは

ジャワ島南岸に突き出す半島の町。バンドンから南西に約215km、車で5〜7時間。ジャカルタやバンドンの都市住民にとっての「近場のビーチリゾート」だが、外国人旅行者にはほとんど知られていない隠れた秘境。Travelfish曰く「平日は静かな漁村、週末とインドネシアの祝日はジャワ随一のビーチリゾートに変貌する」。

### なぜパンガンダランを選ぶか

- **観光客が少ない**: バリやロンボクとは比較にならないほど外国人旅行者が少なく、ローカル感が強い
- **自然の多様性**: ジャングル自然保護区、エメラルドの渓谷（グリーンキャニオン）、サーフポイント、マングローブ林、鍾乳洞が半径30km内に集結
- **抜群のシーフード**: 漁港直結。パサール・イカン（魚市場）で獲れたての魚をその場で炭火焼き
- **コスパ**: 宿泊1泊 Rp 100,000〜300,000（約1,000〜3,000円）、食事 Rp 15,000〜50,000（約150〜500円）

### 地理と気候

| 項目 | 詳細 |
|------|------|
| 位置 | 西ジャワ州南部海岸、インド洋に面する半島 |
| 気温 | 年間平均 27-29℃（常に高温多湿） |
| 乾季 | 5月〜9月（ベストシーズン。晴天多し） |
| 雨季 | 10月〜4月（午後にスコール頻発。グリーンキャニオン閉鎖の可能性あり） |
| 湿度 | 年間を通じて80%前後 |
| 海水温 | 26-29℃ |

### 歴史的背景 — 2006年津波

2006年7月17日、M7.7の地震によりパンガンダランに津波が襲来。最大遡上高8.5m、死者668名、行方不明者45名、避難者3,739名という大災害となった。1921年にもM7.5の地震で津波被害があり、パンガンダランはメガスラスト帯に位置する。

現在は **UNESCO津波レディ認定（2022年）** を取得。津波ハザードマップ、避難経路標識、集合場所標識、避難ビル（TES: Tempat Evakuasi Sementara）が整備されている。ホテルとの連携による避難シェルター体制も構築済み。

> **旅行者へ**: 宿にチェックインしたら、まず避難経路を確認すること。地震を感じたら高台へ。半島南部の自然保護区内の丘（標高約150m）も緊急避難先。

## 2. ビーチ & 自然保護区

### 2.1 パンタイ・バラット（Pantai Barat / 西ビーチ）

パンガンダラン半島の西側に延びる約2kmの砂浜。メインのツーリストエリア。

| 項目 | 詳細 |
|------|------|
| 特徴 | 比較的穏やかな波（珊瑚礁がブレイクウォーターの役目）。海藻・サンゴ・クラゲなし |
| 遊泳 | **可**（南部よりは安全だが、離岸流に注意） |
| アクティビティ | サーフィンレッスン（Rp 200,000-250,000/日）、乗馬（ビーチ沿い）、ATV、カヤック、ミニ・デルマン（伝統的な馬車） |
| サンセット | **絶好の夕日スポット**。西向きのため毎日インド洋に沈む夕日が見られる |
| 雰囲気 | 宿・レストラン・土産物屋が集中。平日は静か、週末は賑やか |
| 漁船ツアー | 漁師の船をチャーターしてパシール・プティ（白砂ビーチ）へ（Rp 50,000-100,000程度） |

### 2.2 パンタイ・ティムール（Pantai Timur / 東ビーチ）

半島の東側。西ビーチとは歩いて10分程度の距離。

| 項目 | 詳細 |
|------|------|
| 特徴 | 白い砂浜。波が荒い場所が多く、遊泳には不向き |
| アクティビティ | ジェットスキー、バナナボート、ドーナツボート |
| サンライズ | **日の出スポット**。朝の散歩に最適 |
| パサール・イカン | 東ビーチ北端に魚市場（→ 食事セクション参照）|
| 雰囲気 | 西ビーチより静か。ローカル色が強い |

> **Tips**: パンガンダランの独特な魅力は「朝は東海岸で日の出、夕方は西海岸で夕日」を同じ場所から徒歩圏内で楽しめること。

### 2.3 チャガール・アラム・パンガンダラン（Cagar Alam Pangandaran / 自然保護区）

半島南部の530ヘクタールを占める国立公園（正式名：パナンジュン自然保護区）。二次林の熱帯雨林、チーク林、草原が混在し、中央の台地は標高約100m、最高地点約150m。

#### 基本情報

| 項目 | 詳細 |
|------|------|
| 入場料（外国人） | 平日 Rp 210,000 / 週末 Rp 310,000 |
| 入場料（インドネシア人） | 平日 Rp 16,000 / 週末 Rp 21,000 |
| 営業時間 | 07:00-17:00（日没前に退出） |
| 入口 | 西ゲートと東ゲートの2箇所 |
| ガイド料 | Rp 350,000/グループ（4名まで、3時間、平日）/ Rp 400,000（週末） |
| 所要時間 | 2-4時間（コースにより異なる） |

#### 見どころ

- **野生動物**: カニクイザル（長尾マカク）、キジャン（ホエジカ / barking deer）、野生バンテン（牛）、ヤマアラシ、オオトカゲ、サイチョウ（hornbill）、各種チョウ類
- **洞窟**: **グア・ジュパン（Gua Jepang）** — 第二次世界大戦中に日本軍が使用した洞窟。内部は暗いので懐中電灯必携
- **ヒンドゥー寺院跡**: バトゥ・カルデ（Batu Kalde）遺跡
- **パシール・プティ（Pasir Putih / 白砂ビーチ）**: 保護区東側の美しい白砂の入り江。砕けたサンゴでできた白い砂浜。シュノーケリングの名所

#### トレイル情報

保護区内には整備されたトレイルが複数あり、西ゲートと東ゲートから環状で歩ける。ガイドなしでも標識に従って歩けるが、ガイド付きの方が圧倒的に多くの動物を見られる（ガイドは両入口に常駐）。

- **ショートコース（2時間）**: 西ゲート → 洞窟 → ヒンドゥー遺跡 → 東ゲート
- **ロングコース（3-4時間）**: 上記 + 台地横断 → パシール・プティ → 海岸線 → 帰還
- **難易度**: 中程度。整備された道だが、湿気と暑さに注意。水を最低1L持参

#### シュノーケリング

| スポット | 詳細 |
|----------|------|
| パシール・プティ（西ビーチ付近） | 白砂ビーチ。機材レンタル可（ビーチで直接、ライフジャケット込み）。初心者にはガイド付き |
| 東ビーチ海洋公園（Taman Laut Pantai Timur） | 高速ボートで約10分。バトゥ・ブアヤ、バトゥ・ラヤール1&2 を巡る。サンゴ礁と熱帯魚 |
| 料金目安 | シュノーケリングツアー Rp 150,000-300,000/人（ボート・機材込み） |

## 3. グリーンキャニオン（Cukang Taneuh）

**パンガンダラン最大の目玉アトラクション。** チジュラン川（Sungai Cijulang）が石灰岩を浸食して作った翡翠色の渓谷。正式名称は **Cukang Taneuh**（スンダ語で「土の橋」＝ 峡谷上にかかる幅3mの天然の陸橋に由来）。

### 基本情報

| 項目 | 詳細 |
|------|------|
| 場所 | パンガンダランから北西へ約31km（車で約45分）。Kertayasa村、Cijulang地区 |
| 入場料 | Rp 20,000/人 |
| ボートチャーター | Rp 225,000/隻（最大5名乗り） |
| 営業時間 | 07:30-16:00（金曜は13:00から） |
| 所要時間 | ボートツアー往復 約45分〜1時間 |
| 休業 | **雨季に川が増水すると安全のため閉鎖**（特に12月〜2月に注意） |
| GPS | -7.7289, 108.4881（おおよそ） |

### 体験の流れ

1. **チケット購入 & ボート乗り場**: 入口でチケットを買い、ボート乗り場へ
2. **ロングテールボートで川を遡上**: エメラルドグリーンの川をエンジン付き木製ボートで約15-20分遡上
3. **峡谷に到着**: 両側から切り立った石灰岩の崖が迫る。高さ数十メートルの崖にコケやシダが生え、鍾乳石（スタラクタイト）と石筍（スタラグマイト）が見える
4. **岩を登って峡谷の奥へ**: ボートを降り、岩場を歩いて奥の洞窟的なエリアへ。壁面から水が滝のように流れ落ち、木漏れ日が差し込む幻想的な光景
5. **泳ぐ・飛び込む**: 渓谷内のプール状の深い場所で遊泳可。**クリフジャンプ（7-9m）** も可能（自己責任）
6. **ロープを伝って移動**: 川の急流部分にはロープが張られており、それを伝って泳いで移動する場面も

### 光の条件 & 撮影

| 時間帯 | 光の状態 | おすすめ度 |
|--------|----------|-----------|
| 07:30-09:00 | 朝の柔らかい光が峡谷に差し込む。水の色が最も鮮やかなエメラルド | ★★★★★ |
| 09:00-11:00 | 日光がかなり入り、写真映えする。まだ比較的涼しい | ★★★★ |
| 11:00-14:00 | 真上からの光。峡谷内は日陰が多いのでまだ美しいが暑い | ★★★ |
| 14:00-16:00 | 午後の光。雨季は午後のスコールのリスク | ★★ |

> **撮影Tips**: 防水カメラ / GoPro 必携。スマホは防水ケースに入れること。朝一番（7:30の開場直後）がベスト — 混雑前で水も澄んでいる。

### ボディラフティング（オプション）

グリーンキャニオンではボートツアーだけでなく、本格的なボディラフティングも体験できる。

| コース | 距離 | 所要時間 | 料金目安 | 内容 |
|--------|------|----------|----------|------|
| ショートトラック（セミ） | 約5km | 2-3時間 | Rp 175,000-300,000/人 | ボートで入口まで→ 泳ぎ・岩登り・ジャンプ |
| ロングトラック | 約10km | 4-5時間 | Rp 300,000-410,000/人 | フルコース。急流・クリフジャンプ3箇所（5-8m） |

**ボディラフティングの詳細:**
- 川の流れに身を任せて下る。ライフジャケット・ヘルメット・ウォーターシューズは提供される
- **ジャンプスポット3箇所**: 2箇所が5-6m、1箇所が7-8m（スキップ可）
- 緑の熱帯雨林に囲まれ、猿や鳥を見ながら川を下る
- 途中の休憩所でコーヒー・紅茶・カップヌードル・スナックが購入可
- **朝出発推奨**: 午前中に開始しないと日没までに終わらない可能性

### 持ち物チェックリスト

- [ ] 水着（泳げる服装）
- [ ] ウォーターシューズ（岩場で必須。現地で購入・レンタルも可）
- [ ] 防水バッグ（ドライバッグ）
- [ ] 防水カメラ or スマホ防水ケース
- [ ] 日焼け止め（ウォータープルーフ）
- [ ] タオル & 着替え（ボートに置いておく）
- [ ] **現金**（ATM・銀行なし。パンガンダラン市内で下ろしておくこと）
- [ ] 朝食を必ず食べてから行く（冷水で体力消耗するため）

### 雨季の閉鎖について

| 状況 | 対応 |
|------|------|
| 小雨・曇り | 通常通り営業。水はやや濁る |
| 大雨後 | 川が増水し濁流に。**安全のため閉鎖**。当日朝に確認が必要 |
| 連日豪雨 | 数日間閉鎖されることも。雨季（10月〜4月）は事前に宿で状況確認を |

> **注意**: 雨季でも毎日閉鎖されるわけではない。乾季のような澄んだエメラルド色は期待できないが、水量が増えて迫力ある景色になることも。

## 4. バトゥ・ヒウ（Batu Hiu）

「サメのヒレ岩」と呼ばれるドラマチックな海食崖の景勝地。

### 基本情報

| 項目 | 詳細 |
|------|------|
| 場所 | パンガンダランから西へ約22km（車で約35分）。Ciliang村、Parigi地区 |
| 入場料 | 無料（駐車場料金のみ: バイク Rp 7,500 / 車 Rp 19,500） |
| 営業時間 | 常時開放（日中推奨） |
| 所要時間 | 30分〜1時間 |

### 見どころ

- **奇岩**: インド洋の荒波に削られた石灰岩が、サメのヒレのように鋭くそびえる。高い崖の上から見下ろす絶景
- **サンセット**: 西向きの崖からインド洋に沈む夕日を眺められる。パンガンダラン市内のビーチより人が少なく、より劇的な景色
- **周辺の洞窟**: 崖の下には小さな海食洞（海蝕洞窟）がいくつかある。潮が引いた時に探検可能
- **波の迫力**: 断崖に打ちつけるインド洋の大波は圧巻。雨季はさらに荒々しい

> **安全注意**: 柵がない場所が多い。崖の端には近づきすぎないこと。特に濡れた岩は滑りやすい。子連れの場合は十分注意。

### アクセス

パンガンダラン市内からバイク（オジェック）で35分程度。レンタルバイクで自力で行くか、宿で半日ツアーを手配（バトゥ・ヒウ + バトゥ・カラスのコンビが定番）。

## 5. バトゥ・カラス（Batu Karas）

パンガンダランから南西へ約30分。「ジャワの隠れたサーフ・パラダイス」。1970年代から変わっていないような、ゆるやかな時間が流れるサーフ村。

### 基本情報

| 項目 | 詳細 |
|------|------|
| 場所 | パンガンダランから約20km（車/バイクで30-40分） |
| 空港 | チジュラン・ヌサウィル空港（CJN）から約30分 |
| 雰囲気 | バリのクタとは真逆。ヨガスタジオもバーもない。「生のインドネシア」 |
| ターゲット | サーファー、デジタルノマド、静かなビーチを求めるバックパッカー |

### サーフィン

| 項目 | 詳細 |
|------|------|
| メインブレイク | The Point（ポイントブレイク）— 砂底、ゆっくりピーリングする波。**世界で最も初心者に適したサーフポイントの一つ** |
| 波の質 | 温かい熱帯の水、砂底で安全、ゆっくり崩れる波で基礎練習に最適 |
| レッスン料金 | 2時間 US$20以下（ボードレンタル込み）— **インドネシア最安クラス** |
| ボードレンタル | Rp 50,000-100,000/日 |
| ベストシーズン | 乾季（5月〜9月）。雨季も波はあるがコンディション不安定 |
| サーフスクール | Pangandaran Surf School And Camp、USurf Lessons Batukaras、Amazon Surf Camp |

### 食事 & カフェ

バトゥ・カラスの食事は小さなワルン（warung）が中心。近年はおしゃれなカフェも増えつつある。

| 店名 | ジャンル | 特徴 |
|------|----------|------|
| **Kokyo** | カフェ | 日本語で「故郷」。コーヒー・抹茶ドリンクが人気。おしゃれな雰囲気 |
| **Salt Cafe** | カフェ | スムージーボウル、軽食。Kokyoの隣 |
| **House of Sawah** | レストラン・バー | 田んぼを見渡すテラス席。ヘルシーなメニュー。**火曜のタコチューズデー** がイベント化（タコス半額＋マルガリータ半額＋ライブ音楽＋ダンス） |
| **Bayview Seafood** | シーフード | 新鮮なローカルシーフード |
| **Kang Ayi Resto** | スンダ料理 | 家庭的なジャワ料理 |
| **Warung Ma Engkar** | ワルン | **レンコ**（スンダ風ココナッツチキンスープ）が名物。サーフ後の回復食 |
| ビーチのコーヒーショップ | カフェ | サーファーのたまり場。朝から日没まで |

### ナイトライフ

基本的にナイトライフは**ほぼない**。それがバトゥ・カラスの魅力。

- **House of Sawah** での週1-2回のイベント（タコチューズデーなど）が唯一の社交場
- ビーチでの焚き火 + ギター弾き語り + ローカルとの交流が「ナイトライフ」
- ビールは入手可能だが、バリのような深夜営業のクラブはない

## 6. チトゥマン・ボディラフティング（Citumang）

グリーンキャニオンとは別の「グリーンバレー」と呼ばれる渓流でのボディラフティング体験。グリーンキャニオンより**難易度が低く、家族向け**。

### 基本情報

| 項目 | 詳細 |
|------|------|
| 場所 | Sukamanah, Bojong, Parigi, Pangandaran（パンガンダランから約30km） |
| 入場料 | Rp 15,000/人 |
| ボディラフティング料金 | Rp 69,000〜/人（10名以上グループ割引）、通常 Rp 100,000-175,000/人 |

### 体験内容

- 透明度の高い清流を、ライフジャケット＋ヘルメットを装着して流される
- 天然のウォータースライダー的な箇所あり
- 途中に休憩スポット（座って景色を楽しむ）
- 川底が見えるほど水が透明
- 周囲は熱帯雨林に囲まれ、鳥のさえずりが聞こえる

### 安全情報

- 開始前にインストラクターからブリーフィングあり
- ライフジャケット・ヘルメット提供
- **参加不可**: 心臓病、高血圧、てんかん、めまい症、貧血、喘息、糖尿病、妊婦

## 7. その他の見どころ

### 7.1 カラピャック・ビーチ（Pantai Karapyak）

| 項目 | 詳細 |
|------|------|
| 場所 | パンガンダランから約20km（車で30-40分）。Bagolo村、Kalipucang地区 |
| 入場料 | Rp 10,000/人 |
| 別名 | **Paradise Beach**（パラダイスビーチ） |
| 特徴 | 広大な岩礁地帯。干潮時にサンゴ岩の表面が現れ、独特な自然のパターンを形成。徒歩で探検可能 |
| おすすめ | 干潮時がベスト。静けさと自然美を求める人向け。観光客はほぼいない |

### 7.2 マングローブ林（Mangrove Forest Batukaras）

バトゥ・カラス近くのマングローブ林。木道（ボードウォーク）が整備されており、マングローブの生態系を間近で観察できる。カヌー/カヤックでマングローブの水路を巡るツアーもある。

### 7.3 パシール・プティ（Pasir Putih / 白砂ビーチ）

自然保護区内にある白砂のビーチ。砕けたサンゴからなる白い砂と透明な海水が特徴。シュノーケリングの名所で、機材は現地レンタル可能。漁師のボートで西ビーチからアクセス（Rp 50,000-100,000）。

### 7.4 グア・ジュパン（Gua Jepang / 日本軍洞窟）

自然保護区内の洞窟。第二次世界大戦中に日本軍が防御陣地として使用した。内部は暗く、懐中電灯が必要。保護区のガイドツアーに含まれることが多い。

### 7.5 コテージ・インダストリー・ツアー

パンガンダラン周辺の村で半日の手工業見学ツアー。Travelfish推奨。

- **クルプック工場**: えびせんべいの製造過程を見学
- **パームシュガー工場**: ヤシの木からアレンシュガー（椰子砂糖）を作る過程
- **ココナッツ産業**: ココナッツオイル・ココナッツファイバーの加工
- **ローカル市場**: 朝市の活気ある雰囲気
- 料金: 約 Rp 150,000/日（ガイド付き）

### 7.6 グア・ペトルック（Gua Petruk）について

> **注意**: Gua Petruk（ペトルック洞窟）はパンガンダランではなく、**中部ジャワのクブメン（Kebumen）** にある鍾乳洞。パンガンダランとは離れている（車で3-4時間）。パンガンダラン周辺の洞窟としては自然保護区内のGua Jepangが主な見どころ。

| 宿 | 蚊帳（mosquito net）のある宿を選ぶか、持参。AC付きの部屋なら窓を閉められるので比較的安全 |
| 蚊取り線香 | インドネシアのバカール・ニャムック（obat nyamuk bakar）は現地で安く購入可能 |

### 12.5 津波安全対策

| 項目 | 詳細 |
|------|------|
| 避難経路 | 市内各所に **Jalur Evakuasi Tsunami**（津波避難経路）の標識が設置済み |
| 避難場所 | 高台の学校、自然保護区内の丘（標高150m）、協定ホテルの上層階 |
| 警報システム | インドネシア気象庁（BMKG）による津波警報システムが稼働中 |
| 行動指針 | 地震を感じたら → 揺れが収まるのを待つ → 即座に高台へ避難（徒歩で行ける最も高い場所）→ 公式の解除情報が出るまで待機 |
| UNESCO認定 | 2022年に **Tsunami Ready** 認定取得済み |

### 12.6 その他の実用情報

| 項目 | 詳細 |
|------|------|
| トイレ | 公共トイレあり（有料 Rp 2,000-5,000）。トイレットペーパー持参推奨 |
| 水 | 水道水は飲めない。ペットボトルの水を購入（Rp 3,000-5,000/600ml） |
| 電圧 | 220V / 50Hz。プラグはCタイプ（ユーロプラグ） |

### 13.3 日本人旅行者にとってのパンガンダラン

4travel.jpの旅行記によれば：

- **「日本人で知っている人はインドネシア通。実際に訪れた人はさらに通」** — それほど日本人には知られていない
- インドネシア在住の日本人が週末旅行で訪れることはあるが、日本からの旅行者はごく稀
- バリやジョグジャカルタと比べて「観光地化されていない生のインドネシア」を体験できる
- 道路状況について：以前はバンジャール→パンガンダラン間の道路が雨季に凸凹だったが、現在は舗装が改善済み。ただしグリーンキャニオン方面の道はまだ改良途上

### 13.4 グア・ジュパン（日本軍洞窟）について

自然保護区内の **Gua Jepang** は第二次世界大戦中に日本軍が掘った / 使用した防御用洞窟。日本人旅行者にとっては歴史的な意味で訪れる価値がある。保護区のガイドツアーで案内してもらえる。

## 16. 注意事項まとめ

### やるべきこと

- [x] バンドンで十分な現金を引き出してから出発
- [x] 水着・ウォーターシューズ・防水バッグを準備
- [x] 虫除けスプレー（DEET含有）を持参
- [x] 宿チェックイン後に津波避難経路を確認
- [x] グリーンキャニオンは朝一番に行く（7:30開場）
- [x] パサール・イカンで新鮮シーフードを堪能
- [x] 帰りの列車チケット（バンジャール→ジョグジャカルタ）を事前にオンライン予約

### やってはいけないこと

- [ ] ATMがどこにでもあると思わない（中心部にしかない）
- [ ] 雨季にグリーンキャニオンの営業を確認せずに行かない
- [ ] 自然保護区で野生のサルに餌を与えない（攻撃的になる）
- [ ] 断崖（バトゥ・ヒウ等）で柵のない場所に近づきすぎない
- [ ] 強い離岸流のあるビーチで泳がない（東ビーチは遊泳不向き）
- [ ] クレジットカードだけに頼らない（ほぼ現金のみの世界）
