## 1. バンドンとは何か

標高768mの高原都市。年間平均気温20〜25℃で、ジャカルタ（海抜8m・猛暑）と比べて格段に涼しく過ごしやすい。オランダ植民地時代に「東洋のパリ（Parijs van Java）」と称されたアールデコ建築の宝庫。スンダ人の文化的中心地であり、スンダ料理は新鮮な野菜と独特のサンバルが特徴。

### 基本データ

| 項目 | 内容 |
|------|------|
| 正式名称 | Kota Bandung（バンドン市） |
| 州 | Jawa Barat（西ジャワ州）州都 |
| 標高 | 650〜1,050m（市街中心部768m） |
| 気温 | 年間平均 20〜25℃（朝晩17℃前後まで下がる） |
| 雨季 | 10月〜5月（特に12〜2月が多雨、月間降水量200mm超） |
| 乾季 | 6月〜9月（月間降水量50mm程度） |
| 空港 | Husein Sastranegara（BDO）※2025年以降Kertajati（KJT）へ移管進行中 |
| ジャカルタから | 高速鉄道Whoosh 約50分 / 在来線Argo Parahyangan 約3時間 / 車 約2.5〜4時間 |

### 気候の詳細

バンドンは熱帯高地気候で、インドネシアの他の大都市と比べて著しく涼しい。

## 3. カワプティ火口湖（Kawah Putih）完全攻略

### 基本情報

| 項目 | 内容 |
|------|------|
| 正式名称 | Kawah Putih（白い火口） |
| 所在地 | Gunung Patuha、Ciwidey地区、南バンドン |
| GPS座標 | -7.1658, 107.4040（7°09'57.0"S 107°24'14.4"E） |
| 標高 | 2,430m |
| 湖の直径 | 約500m |
| 形成時期 | 約400年前の間欠泉活動で形成 |
| 火山分類 | パトハ山（Gunung Patuha）、休火山（最後の噴火記録は1600年頃） |
| 営業時間 | 07:00〜17:00 WIB（最終入場16:00） |

### 入場料（2025-2026年時点）

| カテゴリ | 料金 | 備考 |
|---------|------|------|
| 外国人（シャトル込み） | IDR 122,000 | オンタンアンティン（シャトルバス）往復含む |
| 外国人（シャトルなし） | IDR 90,000 | 自家用車で火口近くまで行く場合 |
| 外国人 サンライズチケット（8:00前入場） | IDR 143,000 | 早朝の幻想的な霧を狙う人向け |
| インドネシア人（シャトル込み） | IDR 63,000 | 国内料金 |
| KITAS/KITAP保持者 | 国内料金適用 | 居住許可証提示 |
| 駐車場（下部） | IDR 11,000 | 一般駐車場 |
| 駐車場（火口近く） | IDR 150,000 | 自家用車で上まで行く場合 |

### オンタンアンティン（Ontang-Anting）シャトルバス

- **概要**: 下部駐車場から火口湖まで約5kmの道を運行する小型オープンサイドバス
- **運行**: 10人集まり次第出発。最大待ち時間30分
- **車両**: ベンチシート型のオープンバス。風が強いので上着必携
- **所要時間**: 片道約10分
- **チケット**: 入場料に含まれている（外国人IDR 122,000の場合）

### 行き方

#### 公共交通機関の場合
1. バンドン市街 → **Leuwipanjang Terminal**（Trans Metro Bandung バスまたはGrab）
2. Leuwipanjang Terminal → **Ciwidey Terminal**（エルフ/ミニバス、IDR 20,000、約1.5時間）
3. Ciwidey Terminal → **Kawah Putih**（アンコット/ミニバス、IDR 10,000〜15,000、約30分）

#### Grabチャーター / プライベート車
- バンドン市街から約50km、片道1.5〜2時間
- Grab Car チャーター: IDR 350,000〜500,000（往復、待機時間込み）
- ツアー参加: IDR 300,000〜600,000/人（カワプティ + シトゥパテンガン + 茶園セット）

### 安全警告

> **硫黄ガスに関する重大な注意事項**

| 警告事項 | 詳細 |
|---------|------|
| 滞在時間 | **火口湖での滞在は15〜30分に制限すること** |
| 硫黄臭 | 非常に強い。風向きによっては目や喉が痛む |
| N95マスク | 持参推奨。ただし**N95は粒子状物質には有効だがSO2ガスには効果が限定的** |
| 立入禁止者 | 呼吸器疾患・心臓疾患の方、妊婦、乳幼児は**入場不可** |
| 閉鎖の可能性 | 硫黄濃度が高い日は管理者判断で臨時閉鎖あり |
| 火口近接 | 柵を越えて水際に近づくのは絶対禁止 |

### 写真撮影のコツ

| ポイント | 詳細 |
|---------|------|
| ベストタイム | **7:00〜10:00**（霧と軟らかい光が幻想的） |
| 湖の色変化 | 硫黄濃度・天候により白→翡翠色→青緑に変化 |
| 霧の演出 | 早朝は霧が湖面を漂い、神秘的な写真が撮れる |
| 注意点 | 硫黄蒸気でレンズが曇る。レンズクロスを頻繁に使用 |
| 三脚 | 使用可。ただし他の観光客の邪魔にならないこと |
| ドローン | 基本的に使用禁止 |

### 天候対策

- 標高2,430mのため気温は**8〜22℃**。市街地より10℃以上低い
- **必携品**: フリース or ダウンジャケット、長ズボン、滑りにくい靴
- 雨季（10〜5月）は午後にスコールの可能性大。**午前中の訪問が鉄則**
- 濃霧で何も見えない日もある（特に雨季）。天気予報を確認すること
- 乾季（6〜9月）がベストシーズン

## 4. タンクバン・プラフ火山（Tangkuban Perahu）

### 基本情報

| 項目 | 内容 |
|------|------|
| 正式名称 | Tangkuban Perahu / Tangkuban Parahu |
| 意味 | 「ひっくり返った船」（スンダの伝説サンクリアンに由来） |
| 種類 | 成層火山（活火山） |
| 標高 | 2,084m |
| 場所 | Lembang、北バンドン（バンドン市街から北へ約30km） |
| GPS座標 | -6.7597, 107.6000 |
| 営業時間 | 07:00〜17:00 WIB |

### 入場料（2025年時点）

| カテゴリ | 平日 | 週末・祝日 |
|---------|------|-----------|
| 外国人 | IDR 200,000 | IDR 300,000 |
| インドネシア人 | IDR 30,000 | IDR 50,000 |
| 駐車場（車） | IDR 20,000 | IDR 20,000 |
| ガイド料 | IDR 100,000〜150,000 | — |

### 火口（クレーター）

#### Kawah Ratu（ラトゥ火口 = 女王の火口）
- **最もアクセスしやすい火口**。駐車場から車で火口の縁まで行ける
- インドネシアで唯一、車で火口の縁まで行ける火山
- 硫黄の噴気孔が見える。黄色い硫黄の堆積
- 所要時間: 30分〜1時間

#### Kawah Domas（ドマス火口）
- **ラトゥ火口から徒歩30分の下り坂**（ユーカリ林を通る）
- 温泉地帯。足湯可能。泥スパもあり（追加料金）
- **名物: 温泉で茹でた卵**（地元の売り子から購入可能）
- 硫黄臭がラトゥ火口の10倍強い（マスク推奨）
- 営業時間: 08:00〜15:30
- 往復所要時間: 3〜4時間（ゆっくり見学含む）
- ガイド推奨（IDR 100,000〜150,000）

#### Kawah Upas（ウパス火口）
- 一般開放されているがアクセスが限定的
- ガイド同伴が必要

#### Cikahirupan火口
- 見学可能な4つの火口の一つだが、訪問者は少ない

### 注意事項
- **2019年7月に噴火あり**（火口から噴石が飛散）。活火山であることを忘れないこと
- 火山活動レベルは随時変更されるので、訪問前にインドネシア火山地質局（PVMBG）の情報を確認
- 週末は非常に混雑。平日の午前中がベスト
- 土産物の押し売りがある。はっきり断ること

### 行き方
- バンドン市街から車/Grab: 約1時間（IDR 100,000〜150,000片道）
- Lembang経由でアンコットもあるが、観光客には不便

## 6. 茶園（Tea Plantations）

バンドン南部の高地には、オランダ植民地時代から続く広大な茶園が広がる。

### 6-A. マラバル茶園（Malabar Tea Plantation）

| 項目 | 内容 |
|------|------|
| 場所 | Pangalengan地区、バンドン市街から南東へ約40km |
| 歴史 | オランダ植民地時代にK.A.R. Bosscha（ボッスハ天文台と同じ人物）が経営 |
| 特徴 | かつてオランダ王室御用達の茶を生産 |

- **Preanger Planters**: Hunderian家、Holle家、Kerkhoven家、Bosscha家の3つの開拓者一族がジャワの茶産業の基礎を築いた
- マラバル茶園は植民地時代のインドネシア茶産業の象徴的存在
- 現在も広大な茶畑が広がり、散策可能

### 6-B. ランチャバリ茶園（Rancabali Tea Plantation）

| 項目 | 内容 |
|------|------|
| 場所 | Ciwidey地区、Kawah PutihやSitu Patenggangの近く |
| 入場料 | IDR 15,000〜20,000 |
| 所要時間 | 1〜2時間 |

- 茶畑の中を散策できるウォーキングコース
- 茶工場見学: 茶葉の摘み取りから製茶までの工程を見学可能
- 写真撮影スポットとして人気（緑の絨毯のような茶畑の景観）

##### 1955年バンドン会議について
- 1955年4月18〜24日開催
- アジア・アフリカ29カ国が参加（世界人口の54%・15億人を代表）
- スカルノ大統領が開会演説「新しいアジアと新しいアフリカを生まれさせよう」
- ネルー（インド）、周恩来（中国）、ナセル（エジプト）らが参加
- **平和十原則（Dasasila Bandung）** を採択 → 非同盟運動の基礎に
- 博物館にはスカルノの等身大蝋人形、各国語の十原則掲示板、会議場の復元展示
- **無料入場**。歴史好きには必見

- 行列ができることも（特に週末）。開店直後がおすすめ

### 14-F. 高山病について

| 場所 | 標高 | リスク |
|------|------|--------|
| バンドン市街 | 768m | なし |
| タンクバン・プラフ | 2,084m | 低い（ただし呼吸器系持病のある方は注意） |
| カワプティ | 2,430m | 低〜中程度 + **硫黄ガスとの複合リスク** |

- 2,500m未満のため一般的な高山病のリスクは低い
- ただし硫黄ガスと標高の組み合わせで息切れしやすい
- 呼吸器・心臓に持病のある方、妊婦、乳幼児はカワプティ不可
- **対策**: ゆっくり歩く、水分を多めに摂る、長居しない（15〜30分）

### 15-C. バンドンと日本の歴史的つながり
