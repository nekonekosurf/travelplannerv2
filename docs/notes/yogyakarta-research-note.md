## 1. ジョグジャカルタとは何か

### 概要

ジョグジャカルタ（地元では「ジョグジャ」または「ジョグ」と呼ばれる）は、インドネシアで唯一の**現役の王制特別州**である。スルタン・ハメンクブウォノ10世が州知事と王を兼任し、王宮（クラトン）は今も機能する政治・文化の中心だ。

ジャワ島の「文化の首都」として知られ、バリ島に次ぐインドネシア第2の観光地。2つのUNESCO世界遺産（ボロブドゥールとプランバナン）への玄関口であり、古典ジャワ文化（バティック、ガムラン、ワヤン、舞踊）が日常に根付く学生の街でもある。

### 歴史ハイライト

| 年代 | 出来事 |
|------|--------|
| 8-9世紀 | シャイレンドラ朝がボロブドゥール建立、サンジャヤ朝がプランバナン建立 |
| 1755年 | **ギヤンティ条約**：マタラム王国がスラカルタとジョグジャカルタに分裂。スルタン・ハメンクブウォノ1世がジョグジャカルタ王国を建国 |
| 1755-1766年 | クラトン（王宮）建設 |
| 1942-1945年 | **日本占領期**：日本軍はジョグジャカルタ王国に比較的自治権を認め、監視下で機能させた |
| 1946-1949年 | **独立戦争中の臨時首都**：ジャカルタがオランダに占領された際、ジョグジャが共和国の首都として機能 |
| 1950年 | 独立への貢献と引き換えに**特別州**の地位を獲得。インドネシアで唯一の王制統治地域に |
| 2006年5月27日 | **M6.4の大地震**：震源はバントゥル付近。死者5,700人以上、被害額29.1兆ルピア（31億ドル）。プランバナン寺院も一部損壊 |

### 気候（雨季の2月末）

| 項目 | データ |
|------|--------|
| 平均気温 | 27-33°C（湿度高い） |
| 降水量 | 2月は雨季のピーク。午後にスコールが多い |
| ベストシーズン | 5-10月（乾季）。ボロブドゥールの日の出は乾季がベスト |
| 注意 | 雨季でも朝は晴れることが多い。ボロブドゥール日の出は運次第 |

### 世界三大仏教遺跡

ボロブドゥールはアンコール・ワット（カンボジア）、バガン遺跡（ミャンマー）と並ぶ**世界三大仏教遺跡**の一つ。世界最大の仏教寺院でもある。

### 建築構造の解説

ボロブドゥールは丘の上に建てられた**巨大な立体曼荼羅**で、仏教の宇宙観を建築で表現している。

#### 3層の世界観

| 層 | 名称 | 構造 | 仏教的意味 |
|----|------|------|------------|
| **下層** | カーマダートゥ（欲界） | 基壇部分（隠れたレリーフ160面） | 欲望に支配された世界 |
| **中層** | ルーパダートゥ（色界） | 5層の方形テラス | 物質的な形はあるが欲望を克服しつつある世界 |
| **上層** | アルーパダートゥ（無色界） | 3層の円形テラス + 中央ストゥーパ | 形をも超越した悟りの世界 |

#### 数字で見るボロブドゥール

| 項目 | 数字 |
|------|------|
| プラットフォーム | **9層**（方形6層 + 円形3層） |
| ストゥーパ（釣鐘型仏塔） | **72基**（透かし彫り）+ 中央大ストゥーパ1基 |
| 仏像 | 元々**504体**（現存するのは約300体） |
| レリーフパネル | **2,672面**（総面積2,500㎡以上） |
| 仏教石彫としての規模 | 世界最大 |

#### 72基のストゥーパ配置

| 円形テラス | ストゥーパ数 | 透かし穴の形 |
|------------|------------|------------|
| 第1テラス | 32基 | ダイヤ型（菱形） |
| 第2テラス | 24基 | 四角形 |
| 第3テラス | 16基 | 四角形（小さい） |

各ストゥーパ内部には仏坐像が安置されている。触ると幸運が訪れるという言い伝えがあるが、保存のため触れないのがマナー。

#### サンライズ鑑賞のコツ

1. **乾季（5-10月）がベスト**: 晴天率が高く、霧と日の出のコントラストが美しい
2. **雨季（11-4月）**: 曇りの確率が高いが、霧が多い分ドラマチックな景色になることも
3. **日の出時刻**: おおよそ05:30-06:00（季節による）
4. **持ち物**: 防寒着（朝は涼しい）、懐中電灯、虫除け
5. **前日ボロブドゥール村に宿泊すると楽**: ジョグジャ市内からは片道1時間以上

### 写真撮影ルール（2025年〜重要変更）

> **2025年以降、ボロブドゥール寺院内での個人撮影は全面禁止**
>
> - スマートフォン、カメラ、GoPro等すべてのデバイスが対象
> - スタッフが常時監視しており、違反者は警告・退場の対象
> - 寺院の外（敷地内の公園部分）での撮影は可能
> - ストゥンブの丘からの撮影には制限なし

### 服装規定

- 肩と膝を覆う服装が推奨（寺院は宗教施設）
- ショートパンツの場合は入口で**サロン（腰巻き布）**を巻いてもらえる（無料）
- 登頂時は**ウパナットサンダル**に履き替え必須（靴・サンダルは預ける）
- サンライズ時は朝冷えるため長袖推奨

## 5. ジョンブラン洞窟（オプション）

### 基本情報

| 項目 | 詳細 |
|------|------|

| **場所** | グヌンキドゥル県、ジョグジャ市内から南東へ約60km |
| **GPS** | -8.0468, 110.6403 |
| **営業時間** | 08:00-14:00（09:30以降の到着は不可） |
| **入場料** | IDR 500,000/人（ガイド・装備込み） |

### 「天国の光」体験

ジョンブラン洞窟の最大の見どころは、**垂直の洞窟口から射し込む太陽光の柱**（Heaven's Light / Cahaya Surga）。正午前後（10:00-12:00頃）に洞窟底部の古代森林に光が射し込み、神秘的な光景を作り出す。

### ツアーの流れ

1. **07:00**: ジョグジャ市内を出発（所要1.5時間）
2. **08:30**: 到着、受付・ブリーフィング・装備（ヘルメット・ハーネス・長靴）
3. **垂直下降**: ロープを使って**60mの垂直の穴**を降下（スタッフが確保、初心者OK）
4. **洞窟内トレッキング**: 暗闘の中を約250m歩いてグルブッグ洞窟へ
5. **天国の光**: 光が射し込む時間帯に到着するよう調整
6. **上昇**: 再びロープで60m登る
7. 全体で約3-4時間

### 注意事項

- **体力**: 普通の体力があれば大丈夫。ただし閉所恐怖症の人には向かない
- **服装**: 汚れてもいい服、長ズボン必須。長靴は貸し出し
- **カメラ**: 防水対策推奨。暗いためスマホでは撮影困難
- **予約**: 以前はオンライン予約必須だったが、2025年現在は**予約不要**。07:00にジョグジャを出発し、早めに到着すればOK
- **ベスト時期**: 乾季（5-10月）。雨季は水位が上がり危険なため閉鎖されることあり

## 10. 実用情報

### 基本インフラ

| 項目 | 詳細 |
|------|------|
| **ATM** | 至る所にあり。BCA、BNI、Mandiriが主要。マリオボロ通り沿いに多数 |
| **両替** | マリオボロ通りの両替商は手数料が高い場合あり。銀行ATMでの引き出しが最もレート良い |
| **病院** | RS Bethesda（Jl. Jend. Sudirman No.70）、RS Panti Rapih（Jl. Cik Di Tiro No.30）が主要病院 |
| **薬局** | Kimia Farma、Guardianがマリオボロ周辺に複数 |
| **WiFi** | ほとんどのホテル・カフェで無料WiFi。速度は良好 |
| **SIMカード** | Telkomsel、XL Axiataが主要。空港や通信ショップで購入可（パスポート必要）。5GB IDR 50,000程度 |
| **電圧** | 220V/50Hz（日本と異なる。変換プラグ必要：Cタイプ） |

### ジョグジャカルタの特別な地位

- インドネシアで**唯一の王制統治自治体**
- スルタン・ハメンクブウォノ10世（Sri Sultan Hamengkubuwono X）が州知事を兼任
- 2012年の特別法（UU No.13/2012）により、スルタンが終身知事を務める権限が正式に法制化
- クラトン（王宮）は今も政治・文化の中心として機能
- インドネシア独立戦争中（1946-1949年）にジャカルタがオランダに占領された際、ジョグジャが臨時首都として共和国を支えた歴史が特別扱いの根拠

### 地震リスク

- 2006年5月27日のM6.4地震（死者5,700名以上）の記憶が残る
- メラピ火山（Gunung Merapi、2,968m）はジョグジャの北30kmに位置する**世界で最も活発な火山の一つ**
- 2010年の大噴火では300名以上が死亡
- 地震・噴火の際の避難経路を確認しておくこと

### 日本占領時代の歴史

- 1942-1945年、日本軍がジャワ島を占領
- ジョグジャカルタ王国は比較的自治を維持（日本軍の監視下）
- 日本語教育が強制された時期があり、年配者には日本語の断片が残る人もいる
- 独立後、日本占領期に訓練されたインドネシア人兵士（PETA）が独立軍の中核を形成

### 学生の街

- ガジャマダ大学（UGM）をはじめ多数の大学が集中
- 学生人口が物価を下げ、B級グルメ文化を牽引
- 夜遅くまで営業する安い食堂（ワルン）が多い
- カフェ文化が急速に発展中

## 11. 日本人目線の情報

### 日本語ブログからの情報

#### ボロブドゥールについて

- 2025年10月現在、ボロブドゥール遺跡の**登頂・内部見学が可能**（一時期制限されていた）
- 参拝は1時間ごとの交代制で、朝一番は**08:30受付、09:00-10:00の1時間**
- 「8世紀後半〜9世紀前半に建設され、アンコール・ワット、バガン遺跡と共に世界三大仏教遺跡と呼ばれている」
- 「少し霧が出てきて怪しいかなと思いましたが真っ赤な太陽が上がって感激でした」（サンライズ体験者）

#### アクセスに関する日本語情報

- **DAMRIバスでボロブドゥールへ行く方法**: マリオボロ（Bank Indonesia前）から発車。IDR 35,000。「最安！空港からボロブドゥール寺院への行き方」として日本語ブログで紹介
- VELTRAやKKdayで日本語ガイド付きツアーが予約可能（IDR 1,200,000-2,000,000程度）
- 日本語ガイド付きだと遺跡の仏教的意味が深く理解できる

#### ラーマーヤナ・バレエの日本人口コミ

- 「ライトアップされたプランバナン寺院をバックに野外の舞台で見ることができる」
- 「ショーがこんなに素晴らしいとは思っていませんでした」
- 200人以上のプロの舞踊家と古典音楽ガムラン演奏者が参加する壮大なスケール
- 送迎付きツアーでの参加が便利（夜の終演後の帰路が心配な場合）

### 治安情報（日本人向け）

#### 全体評価

ジョグジャカルタの治安は**基本的にとても良い**。王室の存在と宗教の教えがしっかり浸透しているため、ホームレスやストリートチルドレンもほとんど見かけない。

#### 注意すべき点

| リスク | 場所 | 対策 |
|--------|------|------|
| **スリ・ひったくり** | マリオボロ通り、プランバナン寺院周辺 | 貴重品は前面に。バイクひったくりに注意 |
| **置き引き** | 飲食店、空港、ホテルロビー | 荷物は常に身体に接触させる |
| **ぼったくり** | ベチャック、アンドン、土産物店 | 乗車前に値段を決める。値切りは最初の50-60%を目指す |
| **交通事故** | 全域 | バイクの交通量が非常に多い。信号・横断歩道はほぼない。渡る時は手を挙げてゆっくり |
| **服装** | 寺院・市内 | 人口の約90%がイスラム教徒。肩・膝を覆う服装を。寺院ではサンダル・ヒール禁止 |

### 日本からのアクセス

- 成田/関空 → ジャカルタ（ガルーダ直行 約7時間）→ ジョグジャカルタ（国内線 約1時間）
- またはシンガポール/クアラルンプール経由
- ジョグジャカルタ国際空港（YIA / Yogyakarta International Airport）は市内から南西に約40km（2019年開業の新空港）
